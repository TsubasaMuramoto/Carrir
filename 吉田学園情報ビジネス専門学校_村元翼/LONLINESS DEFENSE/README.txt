 < キーボードとマウス >
・移動	  ：ＷＡＳＤキー
・回転	  ：マウスカーソル移動
・弾発射  ：マウス左クリック
・ポーズ  ：Ｐキー

 < ゲームパッド >
・移動	  ：左スティック
・回転	  ：右スティック
・弾発射  ：Ｒ２ボタン
・ポーズ  ：ＳＴＡＲＴボタン

//===========================================================
// ルール説明
//===========================================================
 < 一人用 >
・画面端からランダムに出現し地球に向かってくる敵を、
  弾と体当たりで撃退し時間切れになるまで防衛するゲーム
・敵は撃退するたびにスコアが加算されていき、
  最後にランキングに反映される
  
//===========================================================
// アピールポイント
//===========================================================
・プレイヤーの移動範囲を広くするために
  プレイヤーの移動に合わせて背景や敵などのオブジェクトをずらす
  スクロール処理を行った移動ができるようにしました。
・ミニマップを作成し、スクリーン外の敵の様子やプレイヤーの現在地
  を把握できるようにしました。
